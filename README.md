# aaaahhhh_rocket

叫べ。止まったら爆散する。

マイク音量でロケットを上昇させるブラウザゲーム。叫び続けて宇宙(高度200km)を目指せ。
音量が閾値を下回ると爆散。息継ぎは即爆死。

## 遊び方

1. `index.html` をブラウザで開く
2. 「マイクをテスト」でマイク許可を確認
3. 難易度(弱/中/強)を選んで開始
4. カウントダウン後、「AAAaaahhhh!!!!」と叫び続ける
5. 止まったら爆散。到達高度が記録される

## 難易度

| 難易度 | 閾値 | 説明 |
|--------|------|------|
| 弱 | 0.05 | ささやく程度でOK |
| 中 | 0.12 | 普通の叫び声 |
| 強 | 0.20 | 全力の絶叫 |

## 開発

依存ゼロ。ブラウザで開くだけ。

```bash
mise install   # Bun をインストール
bun test       # テスト実行
```

### 構成

- `js/` — 純粋関数モジュール(`*-core.js`)とDOMラッパー(`*.js`)を分離
- `js/game.js` — ゲーム状態機械と物理
- `js/audio.js` — Web Audio API マイク入力
- `js/audio-data.js` — オーディオ数学(RMS, スムージング)
- `js/render.js` — SVG レンダリング
- `js/view.js` — ビュー計算(空の色, 不透明度)
- `js/aaah.js` — 「AAAAH」テキスト生成
- `js/sfx.js` — 効果音合成
- `js/storage.js` — localStorage ラッパー
- `js/storage-core.js` — スコア管理純粋関数
- `js/config.js` — 全チューニング定数
